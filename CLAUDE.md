# Mhaddolkar Family Recipes — Claude Code Context

## Family Overview
**Cuisines:** Maharashtrian, Karwarian, Indian

## Family Members
- **Mom** (Dombivli): rava thalipeeth, sheera, puran poli. Cooks by feel. No spices in thalipeeth — just rava, onion, coconut. Extra ghee in sheera.
- **Thane Maushi** (Thane): batata bhaaji. Maharashtrian. Simple, precise. Perfectly dry batata bhaaji.
- **Shymal Maushi**: puran poli. Festival dishes. Generous. The puran poli shows up at every big occasion.
- **Goregaon Maushi** (Goregaon): coastal fish curry, simple dishes with rich ingredients. Karwarian. Few ingredients, bold flavour. Never overcomplicates.
- **Baby Maushi**: kheer, Indian sweets. Desserts. Always slightly too much sugar. People always ask for more.

## Working with This Project
- Edit `family-profile.yaml` to add or update family members
- Run `python setup.py` to regenerate all files
- Add exported chat conversations to `conversations/` for additional context
- Open `web/index.html` to use the family-facing app
